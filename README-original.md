# MFactory AI - 智能制造投资分析助手

基于AI的制造业公司投资分析工具，5分钟内生成专业投资研究报告。

## 功能特性

- **智能财报分析**：自动解析制造业公司财报，提取关键财务指标
- **产业链可视化**：构建制造业上下游关系图谱，识别供应链风险
- **AI投资观点**：基于行业数据、政策动向生成个性化投资建议
- **多市场支持**：A股、港股、美股主要制造业公司全覆盖

## 技术栈

- **前端**：HTML5 + TailwindCSS + JavaScript
- **部署**：GitHub Pages（静态托管）
- **AI集成**：DeepSeek API（可配置）
- **数据源**：公开市场数据API

## 快速开始

### 本地运行
1. 克隆仓库
2. 用浏览器打开 `index.html`
3. 输入股票代码（如300124）体验AI分析

### 部署到GitHub Pages
1. Fork本仓库
2. 在仓库设置中启用GitHub Pages
3. 选择 `main` 分支和 `/` 根目录
4. 访问 `https://[你的用户名].github.io/mfactory-ai`

## 使用示例

支持股票代码：
- 300124（汇川技术）
- 002747（埃斯顿）  
- 600519（贵州茅台）
- 00700（腾讯控股）
